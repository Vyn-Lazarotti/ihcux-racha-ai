# ihcux-racha-ai

Dividir contas de luz, internet, aluguel e compras de mercado é sempre um caos para quem mora em repúblicas ou divide apartamento com amigos. Itens esquecidos, cálculos errados e cobranças chatas geram atritos. O aplicativo "RachaAí" será focado em simplificar essa vida.

O aplicativo permitirá que usuários:

1. Grupos de Gastos: Criar grupos para "Casa", "Viagem" ou "Churrasco".
2. Divisão Inteligente: Adicionar uma conta e escolher quem paga (divisão igual, percentual ou por consumo).
3. Leitura de NF: Tirar foto da nota fiscal para extrair os valores (foco na interface da câmera).
4. Lembretes de Pagamento: Sistema de "Cutucada" (Nudge) para quem esqueceu de pagar.
5. Saldo Geral: Visão clara de "Quem deve para quem" de forma simplificada.

Este protótipo de baixa fideliade feito no Miro possui 5 telas principais:
1. Dashboard de Grupos: Listagem dos grupos ativos e saldo consolidado.
2. Detalhes do Grupo: Fluxo de gastos específicos de uma casa/viagem.
3. Adicionar Gasto: Interface de formulário para inserir valor, descrição e pagantes.
4. Câmera / Scanner: Interface de como o usuário interage para ler uma nota fiscal.
5. Extrato de Dívidas: Tela focada em quem deve quanto para quem, com botão de "Pagar".
